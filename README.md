# Grafo em C - Estruturas de Dados Avançadas

## Descrição

Este projeto foi desenvolvido em linguagem C para demonstrar a manipulação de grafos utilizando uma **matriz de adjacência**.

O programa permite criar um grafo informando a quantidade de vértices e realizar operações como inserção e remoção de arestas, visualização da matriz, cálculo dos graus dos vértices e listagem das adjacências.

O objetivo principal é aplicar conceitos de estruturas de dados avançadas, utilizando matrizes, funções e manipulação de dados em memória.

---

## Funcionalidades

O sistema possui um menu interativo com as seguintes opções:

### 1 - Limpar grafo
Remove todas as conexões existentes na matriz de adjacência.

### 2 - Inserir/remover aresta
Permite adicionar ou remover uma conexão entre dois vértices.

### 4 - Mostrar graus dos vértices

Calcula a quantidade de conexões existentes em cada vértice.

### 5 - Mostrar adjacências

Mostra quais vértices possuem conexão.

### 6 - Sair

Finaliza a execução do programa.
